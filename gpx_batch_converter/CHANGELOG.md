# Changelog

All notable changes to GPX Batch Converter are documented here.

## 1.2.1 — 2026-07-14

- Added complete QGIS Plugin Repository metadata:
  homepage, repository, tracker and changelog.
- Updated the plugin description and About metadata.
- Documented the bundled GDAL command-line requirements.
- Added a public maintainer contact address.
- Added a minimal GPX test dataset to the source repository bundle.

## 1.2.0 — 2026-07-14

- Added a structured results panel.
- Added CSV export of conversion results.
- Added ESRI Shapefile, GeoPackage, GeoJSON, KML and CSV/WKT output.
- Added background conversion through `QgsTask`.
- Added safe cancellation of the active GDAL process.
- Added source provenance fields in merged outputs.

## 1.1.0 — 2026-07-14

- Added merged outputs by GPX layer type.
- Added configurable merged output prefixes.
- Improved result classification.

## 1.0.3 — 2026-07-14

- Declared QGIS 3.28–4.99 compatibility.
- Added Qt 6 support.
- Fixed the completion notification error.

## 1.0.0 — 2026-07-14

- Initial release.
